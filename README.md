# Book Reports

I don't remember books. Here is as good a place as any to stick some refreshers for when I want to read the sequel.

## Books

- **[Daughter of the Bone Forest](books/daughter_of_the_bone_forest/)** by Jasmine Skye -- YA Fantasy, Book 1 of a series

## Structure

Each book gets its own folder under `books/` with:
- `index.md` -- Overview and links
- `summary.md` -- Full plot summary with character links
- `characters.md` -- Character reference with anchor IDs
- `world.md` -- Magic systems, terminology, locations

## GitHub Pages

This repo is set up to deploy as a site via GitHub Pages using the [Just the Docs](https://just-the-docs.com/) theme. To enable it:

1. Go to **Settings > Pages** in this repo
2. Set source to **GitHub Actions** (or deploy from `main` branch)
3. The site will be available at `https://<username>.github.io/book-reports/`

All markdown files are also fully readable directly on GitHub without the site.
